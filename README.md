# CalculatorApp
### A simple calculator application works on both web and phone
[![My Skills](https://skills.thijs.gg/icons?i=js,react)](https://skills.thijs.gg)

#### Function:
   - Must-have button (AC, DEL, ...)
   - Basic calculate (plus, substract, multiple, divide)

   - Calculation history
   - Search
